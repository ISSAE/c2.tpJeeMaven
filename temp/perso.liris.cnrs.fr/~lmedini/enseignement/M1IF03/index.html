Université Claude Bernard Lyon 1 - M1 informatique

M1IF03 - Conception d'applications Web

Responsable : Lionel Médini

Objectifs pédagogiques

Ce module s'adresse à des étudiants ayant déjà pratiqué les langages à balises (HTML, XHTML, XML) et un peu de scripting côté serveur (PHP). Il se concentre sur les grands principes de conception et de développement d'applications Web. Le parti pris est de considérer le Web comme un medium d'interaction avec les applications que vous concevez, à la fois en termes d'interface et de style de programmation. Ce module est donc très lié au module de gestion de projet et génie logiciel (M1IF01).
Il comporte deux grandes parties :

  • Technologies côté serveur
    • Configuration serveur
    • Programmation côté serveur (CGI, PHP, servlet, JSP)
    • Initiation aux services Web (REST)
    • Frameworks côté serveur (Spring)
  • Technologies côté client
    • JavaScript
    • DOM
    • AJAX (XmlHttpRequest)
    • Bibliothèques côté client (JQuery)

    Cette UE est une composante du projet "multimif" pour les non-alternants. Elle se prolonge au second semestre par l'UE Programmation Web avancée et mobile (M1IF13).
